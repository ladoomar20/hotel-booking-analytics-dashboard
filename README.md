# Hotel Booking Analytics Dashboard (Power BI)

This project analyzes hotel booking operations using an interactive Power BI dashboard.

The dataset represents a hotel booking management system including customer, booking, room, payment, staff, and service tables.

## Dashboard Insights
- Total Revenue
- Total Bookings
- Total Customers
- Revenue by Payment Method
- Room Availability by Type
- Services Requested
